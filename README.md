# 🎯 Игра "Предсказание Событий"

AI-powered игра для предсказания спортивных событий в реальном времени на базе FastAPI и WebSockets.

## 🎮 Возможности

- **3 вида спорта**: Футбол ⚽, Бокс 🥊, CS:GO 🎮
- **Предсказания в реальном времени**: Предсказывайте события и зарабатывайте очки за точность
- **AI-генерация контента**: Динамические события и комментарии через Gemini/Groq
- **Боты-соперники**: Соревнуйтесь с 3 AI-ботами разного уровня
- **Живые анимации**: Уникальные анимации для каждого вида спорта
- **Таблица лидеров**: Рейтинг в реальном времени
- **Русский язык**: Полная поддержка русского языка

## 🚀 Быстрый старт

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте AI (опционально)

Скопируйте `.env.example` в `.env` и добавьте API ключ:

```bash
# Основной: Gemini (рекомендуется)
GEMINI_API_KEY=ваш_ключ_gemini

# Резервный: Groq (бесплатно)
GROQ_API_KEY=ваш_ключ_groq
```

Получите ключ Gemini: https://aistudio.google.com/apikey  
Или бесплатный ключ Groq: https://console.groq.com/keys

Игра также работает без AI, используя предустановленный контент.

### 3. Запустите сервер

```bash
uvicorn main:app --reload
```

### 4. Играйте

Откройте http://localhost:8000 в браузере

## 🎯 Как играть

1. **Выберите вид спорта**: Футбол, Бокс или CS:GO
2. **Введите ваше имя** и начните игру
3. **Смотрите трансляцию** и предсказывайте, когда произойдут события
4. **Выберите тип события** и время (в секундах)
5. **Зарабатывайте очки** за точность предсказания:
   - ±2 секунды: 100 очков
   - ±5 секунд: 50 очков
   - ±10 секунд: 25 очков
6. **Соревнуйтесь** с ботами и другими игроками в таблице лидеров

## 🔍 API Endpoints

- `GET /` - Главная страница игры
- `GET /api/leaderboard` - Таблица лидеров (JSON)
- `GET /api/health` - Статус сервера и AI провайдеров
- `WS /ws/game/{game_id}` - WebSocket для игры

## 🤖 Интеграция с AI

События и комментарии в игре генерируются при помощи LLM.

Игра поддерживает два AI-провайдера:

1. **Google Gemini** (Основной, рекомендуется)
   - Высокое качество генерации
   - Отличная поддержка русского языка
   - Модель: gemini-3-flash-preview
   - Получить ключ: https://aistudio.google.com/apikey

2. **Groq** (Резервный, бесплатный)
   - Бесплатно и быстро
   - Модель Llama 3.3 70B
   - Хорошая поддержка русского языка
   - Получить ключ: https://console.groq.com/keys

Подробная инструкция в `AI_SETUP.md`.

## 🛠️ Технологии

- **Backend**: Python 3.12+, FastAPI + WebSockets
- **Frontend**: Vanilla JavaScript + CSS анимации
- **AI**: Google Gemini / Groq (Llama 3.3)
